# Series Tracker

Projeto desenvolvido em React + Node.js + MongoDB

## Funcionalidades
- Adicionar séries
- Filtrar por estado
- Filtro por letra
- Sistema de rating (0-10)

## Tecnologias
- React (Vite)
- Node.js / Express
- MongoDB

## Como correr

### Frontend
cd series_tracker
npm install
npm run dev

### Backend/API
cd backend
npm install
npm run dev
